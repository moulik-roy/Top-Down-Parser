Make sure MINGW and Flex are installed on your Windows Operating System.
All the required files Terminals.txt, CFGLeftRecursionRemoved, Lexical_Analyzer.l and main.cpp should be in the same folder.

1. Open Command Prompt.

2. Compile the Lex file using the following command
flex Lexical_Analyzer.l

3. Compile the C file lex.yy.c generated by LEX along with the C++ file using the following command
g++ lex.yy.c main.cpp

4. Execute the executable file using the following command
a.exe
